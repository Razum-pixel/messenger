О проекте

“Гостевой чат университета” — простой мессенджер для гостей без регистрации: подключились, выбрали имя и общаетесь в общем канале.

Стек технологий

  • Spring Boot
  • WebSocket + SockJS
  • STOMP (Simple Broker)
  • Чистый HTML / Tailwind CSS / JavaScript
